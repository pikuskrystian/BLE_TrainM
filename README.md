# BLE_TrainM

BLE_TrainM is an multi platform appliacation for controling multiple lego wirless train at the same time.

## Technologies

Project is created with:
* QML
* QMake
* C++

### Hardware
* LEGO Powered Up Hub 88009
* LEGO Engine 88011
